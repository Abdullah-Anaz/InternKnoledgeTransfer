GitHub is a Git repository hosting service that provides a web-based graphical interface. It is the world's largest coding community.

What is a repository?

A repository is a project containing files and folders. A repository tracks versions of files and folders.

What is a branch?:

A branch is a parallel version of your repository. By default, your repository has one branch named main and it is considered to be the definitive branch. Creating additional branches allows you to copy the main branch of your repository and safely make any changes without disrupting the main project.
